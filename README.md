# Stored Procedures and Functions
A beginner-friendly SQL project demonstrating how to create reusable logic using stored procedures and functions. It covers simple examples using parameters and return values to help understand modular SQL programming.

## Features
- One stored procedure with input parameter.
- One function returning a value.
- Simple employee table.
- Works on MySQL workbench

## Technologies Used
- MySQL Workbench
- SQL (DDL, DML, Stored Procedure, Functions)

## Steps to run
- Create the database "Procedures_db" using the provided SQL script.
- Run the table creation and insert commands.
- Execute the stored procedure "CALL GetHighSalary(value);".
- Run the function using "SELECT TotalEmployees();".
- View results directly in your SQL script.
